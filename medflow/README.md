# MedFlow - Healthcare AI Agent Workflow Builder
## FastAPI + HTMX Prototype

A no-code workflow builder for healthcare AI agents.

### Setup

```bash
# Create virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run the app
uvicorn main:app --reload --port 8001
```

### Features

- Visual workflow builder for healthcare AI agents
- HIPAA-compliant data handling patterns
- Clinical note analysis
- Patient data summarization
- Medical literature search integration
- Audit logging for compliance

### Use Cases

- Automate clinical documentation
- Patient intake processing
- Medical coding assistance
- Clinical trial matching
- Prior authorization workflows
